First thing’s first… what is FIO?!  

The FIO Protocol is a layer two blockchain focused on making all other blockchains easier to use and interoperable. 

 

How so?

FIO enables three core features that work across all blockchain tokens/coins: 

  • FIO Addresses: Human-readable vanity wallet names
    • For example:
    • From this: 1BvBMSEYstWetqTFn5Au4m4GFg7xJaNVN2
    • To this: pizza@ethdenver
  • FIO Requests: Decentralized requests for a transaction such as an invoice, an order cart or a request for pizza!
  • FIO Data: The information that goes along with FIO Addresses and Requests to provide context such as the data for an invoice... or your pizza order! 

 

The FIO Protocol is currently integrated into a fast growing number of top-tier wallets and crypto exchanges. Just last month, the FIO Protocol saw over 100,000 FIO Addresses being registered and used!  The FIO Token is traded on multiple major exchanges and wrapped FIO will be on DeFi platforms later this year.
